# TAPTOGEN.COM — ULTIMATE ANTIGRAVITY MASTER PROMPT

You are Antigravity acting as a world-class principal product architect, Astro/Tailwind engineer, UX strategist, SEO strategist, technical content strategist, and conversion-focused SaaS designer.

Your mission is to build TapToGen.com as a premium, luxury, ultra-fast, fully responsive, SEO-first generator tools platform. The goal is not to create a normal tools website. The goal is to create the best generator tools website in the world: visually premium, functionally smooth, technically clean, scalable, AdSense-safe, AI-search-visible, and designed so competitors do not feel close in quality.

This prompt replaces an older calculator-site prompt. Do not build a calculator-only website. Build a universal generator tools platform based on the TapToGen brand and the complete tool keyword list provided below.

======================================================================
1. BRAND AND PRODUCT VISION
======================================================================

Site name: TapToGen
Domain: TapToGen.com
Core meaning: Tap to Generate
Primary niche: Online generator tools, AI writing tools, text generators, font/text style generators, name generators, username generators, tag/hashtag generators, social media tools, SEO generators, creative generators, random/fun generators, and utility generators.

Target users:
- Bloggers, creators, freelancers, marketers, students, social media users, gamers, small businesses, developers, and general internet users.
- Users who want fast, free, simple, browser-based generation tools.
- Users who need instant outputs without complicated signup flows.

Core promise:
TapToGen helps users generate useful text, names, styles, tags, captions, usernames, bios, ideas, and digital assets instantly with clean, easy, premium tools.

Positioning:
TapToGen should feel like a premium AI/SaaS product, not a basic free-tools directory.

2. CRITICAL STRATEGIC RULES
======================================================================

You must follow these rules strictly:

1. Every tool page must match the exact search intent of its primary keyword.
2. Do not create thin duplicate pages. Similar keywords must be clustered intelligently.
3. Each tool must feel useful, unique, and purpose-built for that intent.
4. Every tool page must include an actual working generator, not just content.
5. Every generator must have a clean UI, strong UX, copy buttons, reset buttons, examples, smart placeholders, and helpful output formatting.
6. Every page must include human-written, SEO-optimized, semantically rich content below the tool.
7. Content must be AdSense-safe, family-safe, helpful, original, and not spammy.
8. Do not use placeholder content, dummy pages, TODOs, or broken routes.
9. Build a scalable architecture so hundreds or thousands of tools can be added smoothly.
10. The design must be luxury, premium, responsive, fast, beautiful, and unique.
