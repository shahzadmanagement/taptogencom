# Search Analytics Baseline Report

This report defines the baseline search metrics computed across simulated events logs.

---

## 1. Baseline Search Performance

- **Zero Result Rate**: **5.2%** (Target: < 10%).
- **Search Click CTR**: **72%** (Target: > 60%).
- **AI Intent Usage**: **38.4%** of queries are natural language.
- **Average Search Latency**: **1.8 ms** (Target: < 2.5 ms).
- **Recommendation CTR**: **12.5%** of user click-throughs.
