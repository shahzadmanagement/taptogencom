# Google Rich Results Target Guide

This document details key rich snippet targets, requirements, and compliance guidelines.

---

## 1. Rich Results Target Specs
- **SoftwareApplication / WebApplication**: Verified standard category attributes and pricing objects.
- **FAQPage**: Maps active list items directly to schema entries.
- **BreadcrumbList**: Follows strict ordered list nesting constraints.
- **HowTo / ImageObject**: Outputs step arrays and resolved image location properties.
