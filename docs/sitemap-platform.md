# Enterprise XML Sitemap Specifications

This document defines sitemap index files structure, namespace compliance, indexable route configurations, and Google Search Console indexing guidelines.

---

## 1. Quality & Performance Rules
- **Canonical Pathways Only**: Enforces canonical HTTPS urls in sitemaps (nolocalhost/preview entries).
- **Indexable Filtration**: Prevents inclusion of noindex elements or 404 targets.
- **Multilingual Support**: Maps regional tool detail pages separately to prevent search loops.
- **Namespace Compliant Image Sitemap**: Follows standard Google XML schemas for image attributes.
