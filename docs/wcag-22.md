# WCAG 2.2 AA Compliance Checklists

This document details key accessibility checks, contrast ratios requirements, and motion controls guidelines.

---

## 1. WCAG 2.2 Success Criteria
- **Contrast Ratios**: Minimum contrast ratio of $4.5:1$ for normal text, $3:1$ for large text, verified across light and dark modes.
- **Keyboard Traps**: Guarantees users can enter and exit all fields using keyboard Tab keys only.
- **Motion Reduction**: Applies CSS media query `@media (prefers-reduced-motion: reduce)` to disable non-critical animations.
