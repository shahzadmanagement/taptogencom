# Enterprise Metadata Specifications

This document defines title and meta description constraints, standard regional attributes, and formatting checks for search indexing.

---

## 1. Metadata Constraints
- **Title Layouts**: Append brand identifier `— TapToGen` where missing.
- **Description Target Length**: Enforce target lengths between 140 and 160 characters.
- **Dynamic Robots tag**: Maps `noindex` slugs and templates to `noindex, follow` directives.
- **Open Graph / Twitter Sync**: Synchronizes title and description tags across all social platforms.
