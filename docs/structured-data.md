# Structured Data Engine Integration Guide

This document defines core API parameters, compiler routines, and layout integrations.

## 1. Engine Integrations Specs
- **Dynamic generator function**: `buildSchemas(options)` compiles structured schema lists dynamically.
- **Vite & Astro integrations**: Defer elements printing inside base layout scripts.
- **Custom schemas extensions**: Injected via options properties mapping to custom target layouts.
