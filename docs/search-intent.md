# Search Intent Mapping Specifications

This document defines target user intent clusters and search queries optimization rules.

---

## 1. Intent Profiles
- **Informational**: Addressed via "What is this tool?" and "Why use this tool?".
- **Transactional / Operational**: Addressed via step-by-step guides, interactive setting descriptions, and CTA references.
- **Navigational**: Addressed via related links and categories explore matrices.
- **Investigational**: Addressed via Best Practices and Common Mistakes details.
