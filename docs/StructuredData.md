# Structured Data Schema Standards

This document outlines JSON-LD rules mapped inside TapToGen layout pages.

---

## 1. WebApplication Schema

WebApplication schema tags map active tools using parameters:
* `name`
* `description`
* `url`
* `applicationCategory`
* `browserRequirements`

---

## 2. FAQPage & Breadcrumbs

* **`FAQPage`**: Standardizes FAQ questions and answers to show expandable rich results on Google search pages.
* **`BreadcrumbList`**: Lists hierarchical folder paths to make navigation paths clear.
