# Content Component Architecture

This document describes modular content segments created to optimize descriptions, FAQs, and navigation.

---

## 1. Modular Content Components

* **`IntroSection.astro`**: Renders semantic what-is descriptions.
* **`ExamplesSection.astro`**: Generates sample use-case tables dynamically.
* **`FaqSection.astro`**: Groups diagnostic details under translation-safe dropdown summary selectors.
* **`RelatedSection.astro`**: Displays other tools inside the same category layout dynamically, standardizing internal linking.
