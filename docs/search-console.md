# Google Search Console & Bing Webmaster Verification Guide

This document details verification methods, XML site mappings, and ownership properties.

---

## 1. Domain Ownership Verification
- **HTML Meta Verification Tag**:
  - Google: `<meta name="google-site-verification" content="google-mock-verification-id-1234" />`
  - Bing: `<meta name="msvalidate.01" content="bing-mock-verification-id-1234" />`
- **Sitemap Mappings**: Point Search Console directly to dynamic index `/sitemap-index.xml`.
