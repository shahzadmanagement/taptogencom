# Search SEO Intelligence Platform

This document describes the automated checks for keyword cannibalization, landing page gaps, and internal linking opportunities.

---

## 1. Landing Page Gap Checker

The engine scans search logs to identify traffic opportunities:
- **Missing Landing Pages**: Flags zero-result queries that have search volume but lack dedicated tool pages.
- **Urgent Action**: Suggests specific URLs like `/tools/custom-generator-name/` to build and capture search intent.

---

## 2. Keyword Cannibalization & Intent Auditing
- **Overlap Detector**: Scans primary/secondary keywords across all 430+ tools. If multiple tools declare the exact same keyword, flags them to prevent search engine confusion.
- **Link Builder**: Suggests internal cross-linking options to guide users from broad hubs to niche child pages.
