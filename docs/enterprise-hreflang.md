# Enterprise Hreflang & International SEO Specs

This document defines alternate mappings, locale codes configurations, x-default requirements, and dynamic regional target mapping rules.

---

## 1. Hreflang Alignment Specifications
- **Locale Mappings**: Covers all 17 translation codes plus default English version.
- **x-default Attribute**: Targets the default English page for all non-mapped regions.
- **Self-referential Alternate**: Correctly includes matching self link references to pass Search Console checks.
- **English-Only Routing Fallback**: Non-translated pages (static pages, category details, hubs) fall back to self-referential English and `x-default` configurations.
